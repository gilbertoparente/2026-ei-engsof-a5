## 1. Criar a Base de Dados
Executar o script SQL disponível na pasta `database` para criar a base de dados no servidor PostgreSQL.

## 2. Inserir Dados de Teste
Após a criação da base de dados, executar o script de inserção de dados de teste.

## 3. Configurar a Ligação à Base de Dados
Configurar a *connection string* no ficheiro `appsettings.json` com os dados do servidor PostgreSQL local.
